# Python Snake Game 1.0

A simple, classic Snake game built with Python and the Pygame library. This project was developed as part of a learning journey into Object-Oriented Programming (OOP) and Cybersecurity fundamentals, focusing on collision logic, state management, and clean code structure.

## Features
* **Main Menu**: Interactive start screen with Start and Quit options.
* **Scoring System**: Real-time point tracking as you play.
* **Core Mechanics**: Smooth snake movement, growth upon eating apples, and collision detection (walls and self).
* **Grid-Based Spawning**: Apples spawn on a 20px grid to ensure precise alignment with the snake's head.

## Requirements
* Python 3.x
* Pygame 2.6.1
